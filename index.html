What's included in the boilerplate

Everything configured, nothing to wire up manually.

Sanctum Authentication

Register, login, logout, token refresh, and authenticated user endpoint. All form requests with validation included.

Spatie Roles & Permissions

Pre-configured with Admin, Editor, User roles. Seeders ready. Role middleware attached to protected routes.

Rate Limiting

Per-user and per-IP throttling. Separate limits for auth routes (stricter) and general API routes. Configurable in one file.

API Versioning

Clean v1/v2 route structure. Controllers separated by version. Clients using v1 won't break when you ship v2.

Consistent Error Handling

Global exception handler returns uniform JSON for validation errors, 404s, auth failures, and server exceptions.

Postman Collection

Pre-built requests for every endpoint with environment variables. Import and start testing in under a minute.
